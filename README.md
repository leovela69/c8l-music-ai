# C8L Music AI | Premium Studio

Este proyecto es una plataforma de generación de música impulsada por IA, construida con **Vite + React** para el frontend y **Express** para el backend.

## 🚀 Cómo empezar

Debido a restricciones de seguridad en mi terminal de Windows, no pude ejecutar el comando de instalación por ti. Por favor, sigue estos pasos:

1. **Abre tu terminal** (PowerShell o CMD) en la carpeta:
   `C:\Users\User\Desktop\C.8.L. MUSIC AI`

2. **Instala las dependencias**:
   ```bash
   npm install
   ```

3. **Inicia el entorno de desarrollo**:
   ```bash
   npm run start
   ```
   *Esto iniciará tanto el frontend (Vite) como el servidor (Express) simultáneamente.*

## 🛠 Tecnologías
- **Frontend**: React 18, Vite, Framer Motion (animaciones), Tailwind CSS, Lucide Icons.
- **Backend**: Node.js, Express, TSX.
- **Diseño**: Quantum Industrial (Premium Dark Mode).
